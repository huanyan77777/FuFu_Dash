# 这是什么

fufu 快跑————基于 arduino 的复古音游 demo

软工的特色 arduino 课程，没有什么意思，不如在最后验收的时候搞个小组作业整点活（果真是小组作业吗）

这课最后给的分也没有很高啊喂！

### 模块：

- 雪未来 fufu 2024 版（竟然只要 300r 就拿下了）
- 开发板
- 红外接收装置和它的遥控器
- 一个 2*16 的 lcd 屏幕
- DFPlayer mini mp3 和它的喇叭
- tf卡（fat32）

### 音乐：

深海少女 (1:07 之前)

### 碎碎念

- 没什么用的复古小游戏实现  
- 代码也没什么参考价值（史），时间过于紧张加之应付个小- 作业出来，大部分时间桓衍都用来教 fufu 打音游了（自信音游 fu ×）  
- 为什么叫fufu快跑，致敬我们传奇的二指（？）音游 Muse Dash !  
- 可以送桓衍两首歌吗！
